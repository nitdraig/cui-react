# Inputs

This component renders an input field with a customizable label.

## Import

```tsx
import {Inputs}  from "cui-react";
```

## Example
Render an input field:
<div className="flex items-center rounded-md bg-slate-100 h-32 p-4 space-x-4 ">
 <Input id="username" name="username" labelText="Username" type="text" /></div>


 ## Customizing
You can customize the Inputs component using the following props:

- id (required): ID for the input field.
- name (optional): Name attribute for the input field.
- labelText (optional): Label text for the input field.
- type (optional): Type of input field (e.g., "text", "email", "password").
- pattern (optional): Pattern attribute for input validation.
